Locate anyone with just a link
# install 
termux 
```
pkg install git
pkg install wegt
pkg install php
git clone https://github.com/mrvanonmus/detect_location
```

```
apt-get install git
apt-get install wegt
apt-get install php 
git clone https://github.com/mrvanonmus/detect_location
```
#start
```
cd detect_location
chmod +x *
./1.sh
```
